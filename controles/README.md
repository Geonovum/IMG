# Controles

Deze map bevat rapportage scripts waarmee je bepaalde aspecten van het model kunt controleren. Er zijn twee scripts:

- **mim2report.xsl**: maakt een rapportage vanuit het MIM exchange formaat.
  Momenteel wordt alleen een rapportage gemaakt waarin alle kenmerken te vinden zijn. Hiermee is
  bijvoorbeeld snel te zien of het patroon veld overal goed is ingevuld.
- **xsd2report.xsl**: maakt een rapportage vanuit het XSD. Hiermee is
  bijvoorbeeld snel te zien of het patroon veld overal goed terechtkomt in het xsd.
- **Makefile**: bestand met workflows. Kan op linux systemen gebruikt worden
  om de rapportages automatisch te genereren.
- **geonovum.css**: stylesheet voor de gegenereerde rapporten.
