# Testbestanden voor het imgeluid xsd

In deze map vind je een aantal testbestanden voor het xml schema. Deze bestanden
zijn gekopieerd van <https://github.com/rivm-syso/CVGG/tree/main/voorbeeldbestanden>
en daarna aangepast.

Enkele opmerkingen:

- Als testset is deze lijst bestanden zeker niet volledige.
- Ook zijn het geen voorbeeldbestanden omdat het in sommige gevallen juist de
  bedoeling is dat een fout gerapporteerd wordt.
