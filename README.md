# IMG

Deze repository bevat het Informatie Model Geluid.

De laatste officieel gepubliceerde versie van het model is te vinden via
<https://docs.geostandaarden.nl/cvgg/img/>

De laatste werkversie van het informatie model is te lezen via GitHub Pages:
<https://geonovum.github.io/IMG/>

Issues zijn te vinden via <https://github.com/Geonovum/IMG/issues>

Geonovum wil zelf het beheer houden op de issues, dus als je een nieuw issue wil
indienen, verzoeken we dat te doen via de helpdesk (img@geonovum.nl).
Dan zet Geonovum het op Github. Daarna is iedereen vrij om er op te reageren.
